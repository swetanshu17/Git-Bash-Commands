GIT BASH COMMANDS

  1. To check wethether we are in which folder - pwd
  2. To see what are sub files inside the folder - ls
  3. To go inside a folder - cd "Folder Name"
  4. To go back from any folder or drive - cd ..
  5. TO check the version of the git - git --version
  6. To initalize git in folder - git init
  7. To check wether git file is initillized or not - ls -a
  8. TO delete .git file - rm -rf .git
  9. To commit in git bash - git commit -m "commit message"
  10. To check how the status of commit - git log
  11. To push the code from Git Bash - git push -u origin (branch name)
  12. To check wether u are in which branch - git branch
  13. To create New Branch - git branch "Branch Name"
  14. TO Go to that Branch - git checkout "Branch Name"
  15. TO create branch and go to the branch directly - git checkout -b "Branch Name"
  16. To Stack the changes - git add "File Name"
  17. To Stack all the changes in the file - git add .
  18. To create any file through git bash - touch file_name.file_extension
  19. TO add remote to ur gitbash - git remote add origin (Repo URL)
  20. TO see u are in which remote - git remote show
  21. TO add username in ur git bash - git config --global user.name "username"
  22. TO see ur username - git config --global user.name
  23. TO add useremail in ur git bash - git config --global user.email "useremail"
  24. TO see ur useremail - git config --global user.email
